# 排班与假期管理

## 排班表导入

### 文件格式

排班表为日历格式 Excel，每个 Sheet 一个月份（如 `1月`、`2月`），包含：
- 标题行：年份（如 "2026年"）
- 表头行：周次 | 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日
- 数据行：每行的周一到周日列填入对应日期

### 休息日识别

通过单元格**背景填充色**判断：
- 有填充色（非白色/默认）→ 休息日 (`!isRest` → `workDays[date] = false`)
- 无填充色 → 上班日 (`workDays[date] = true`)

填充色检测逻辑 (`_hasFill`)：
```js
// 排除：patternType === 'none'
// 排除：fill.rgb === 'FFFFFF' 或 'FFFFFFFF'（白色）
// 排除：fill.indexed === 64 或 65（系统默认色）
// 排除：fill.theme === 1 且 fill.tint === 0（无着色）
// 有 fill.rgb 或 fill.indexed 且非排除值 → 有填充色
```

### 按员工展开

排班表从 Excel 读取时不含员工信息。导入时执行展开：
```
排班记录(s) × 员工列表(employees) → schedules 记录
```

每条 schedules 记录包含：
```
{ employeeNo, name, department, year, month, workDays }
```

### 排班表回退

员工可能没有关联排班记录（从打卡数据提取的新员工未在排班表中）。计算时的回退策略：
1. 按 `[employeeNo+year+month]` 精确匹配
2. 若无匹配，查询同年同月的任意一条排班记录作为默认
3. 若完全无排班数据 → 全部默认为上班日

## 假期管理

### 假期记录结构

```
{ date, name, isWorkday, isHoliday }
```

- `isWorkday = true` → 强制上班（调休上班，覆盖排班休息）
- `isHoliday = true` → 强制休息（法定假期，覆盖排班上班）

### 日期段录入

v1.0.10 起支持日期段批量录入：
- 开始日期 + 结束日期 → 自动生成范围内每天一条记录
- 例：2026-01-28 至 2026-02-03 "春节" → 生成 7 条记录

### isWorkDay 判定融合

`_isWorkDay()` 将排班表和假期管理统一判定：

```js
// 1. 假期优先
const holiday = holidaysData.find(h => h.date === dateStr);
if (holiday) {
  if (holiday.isWorkday) return true;    // 调休强制上班
  if (holiday.isHoliday) return false;    // 假期强制休息
}

// 2. 排班表
return schedulesData.workDays[day] === true;  // true=上班, false=休息
```

## 考勤规则配置

通过设置页管理，存储在 `settings.attendance_config`：

```js
{
  workStartTime: '08:30',  // 上班时间
  workEndTime: '17:30',    // 下班时间
  lateThreshold: 0,        // 迟到阈值(分钟，0=无缓冲)
  earlyThreshold: 0,       // 早退阈值(分钟，0=无缓冲)
  graceTimes: 2,           // 容错次数(月)
  graceMinutes: 30         // 容错累计时长(月，分钟)
}
```

修改配置后保存时自动写入 `config_updated_at` 时间戳，考勤页加载时检测并提示用户重新计算。

### 日历视图

`attendance.html:414-415` — 日历单元格背景色：
- 无打卡记录但有 `holidayName` → 红色背景 (`bg-red-50 border-red-300`)

## 在导出中的使用

`excel.js:348-369` — `exportCalendarReport()` 从 DB 读取当月 holidays（`holidaysForMonth`），连同 results、schedules 一并 POST 到 `/api/export/calendar`。

`export_server.py:164-182` — `build_calendar_report()` 接收 holidays 参数：
1. 构建 `holiday_map`（`{date: {name, isWorkday, ...}}`）
2. 对每天，若命中 holiday_map，则 `schedule_label = holiday.name`，`is_rest = !holiday.isWorkday`
3. 否则回退到排班表判定

`export_server.py:189-192` — `schedule_label` 写入导出 Excel 的"排班"列，假期名称直接出现在排班列中。

`export_server.py:189-190, 205-206, 218-219, 230-231` — `is_rest` 控制灰色填充 (`GRAY_FILL`)：休息日的整行（日期、排班、打卡时间、所有员工列）均使用灰色背景。
