# 导出模板系统

## 概述

导出中心 (`export.html`) 提供两种导出模式：
1. **Flat 导出**：传统行列式 Excel，用户自定义字段模板
2. **日历月报导出**：按部门+员工列排布的日历格式月报

## Flat 导出

### 模板设计

用户可在导出中心编辑模板字段：
- 添加/删除列
- 自定义列名
- 从 17 种可用字段中选择数据源
- 支持 `_index`（自动序号列）
- "另存为模板" 保存到 `export_templates` 表

### 可用字段

| 字段名 | 说明 |
|--------|------|
| _index | 自动序号 |
| employeeNo | 考勤号码 |
| name | 姓名 |
| department | 部门 |
| date | 日期 |
| period | 对应时段 |
| scheduleStart | 上班时间 |
| scheduleEnd | 下班时间 |
| signIn | 签到时间 |
| signOut | 签退时间 |
| lateMinutes | 迟到时间 |
| earlyMinutes | 早退时间 |
| overtimeHours | 加班时间 |
| travelHours | 出差时间 |
| absent | 是否旷工 |
| status | 状态 |
| leaveType | 请假类型 |

### 导出流程

1. 用户选择月份（或全部）→ 从 `attendance_results` 查询
2. 按模板字段映射数据
3. SheetJS 生成 `.xlsx` 文件并下载

## 日历月报导出

### 格式说明

按企业月考勤月报模板设计：
- **行结构**：每天 2 行（上午行 + 下午行），整月共 2×N 行
- **列结构**：日期列 + 排班列 + 打卡时间列 + 各部门员工列
- **前 3 列合并**：日期、排班、打卡时间在 2 行中合并为一格

### 表头结构

```
Row 0: 日期 | 排班 | 打卡时间 | 市场营销部 | (空) | 技术部 | ...
Row 1: (空) | (空) | (空)     | 梁XX      | 覃XX | 张三   | ...
```

部门列标题跨员工合并。

### 单元格内容规则

| 条件 | 上午行 | 下午行 |
|------|--------|--------|
| 有打卡签到 | signIn 时间 | signOut 时间 |
| 有 OA 标注 (请假/加班/出差/补卡) | 标注文本 / 拼接 | 下午有签退则填写签退 |
| 休息日 (status=rest) | 空 | 空 |
| 请假 (status=leave) | "请假" | 空 |
| 出差 (status=travel) | "出差" | 空 |
| 缺勤 (absent=true) | "缺勤" | 空 |

### 日期序列化

日期列使用 Excel 序列号（基于 1900 日期系统），通过 `(Date.UTC(...) - Date.UTC(1899, 11, 31)) / 86400000` 计算。

### 排班判定

月报中的排班列通过 schedules 表判定：
- 存在 `workDays[day] === true` 的记录 → "工作日"
- 不存在 → "休息日"
